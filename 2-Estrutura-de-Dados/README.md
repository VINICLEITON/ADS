# 2. Estrutura de Dados

Disciplina que aborda as principais estruturas de dados, sua implementação e complexidade algorítmica.

## Estrutura de Pastas

```
2-Estrutura-de-Dados/
├── exercicios/    # Exercícios práticos resolvidos
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

- Listas, Pilhas e Filas
- Listas Ligadas (Linked Lists)
- Árvores Binárias e AVL
- Grafos
- Algoritmos de Ordenação (Bubble Sort, Merge Sort, Quick Sort)
- Algoritmos de Busca (Linear, Binária)
- Tabelas Hash
- Complexidade de Algoritmos (Big O Notation)

## Linguagens

Python / Java / C (conforme orientação do professor)
