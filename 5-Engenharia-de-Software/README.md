# 5. Engenharia de Software

Disciplina que aborda metodologias, processos e boas práticas para o desenvolvimento de software de qualidade.

## Estrutura de Pastas

```
5-Engenharia-de-Software/
├── exercicios/    # Atividades e estudos de caso
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

- Ciclo de vida do software (SDLC)
- Metodologias: Cascata, Espiral, Ágil (Scrum, Kanban, XP)
- Levantamento e análise de requisitos
- UML: Diagrama de Casos de Uso, Classes, Sequência, Atividades
- Padrões de Projeto (Design Patterns)
- Testes de software (unitário, integração, sistema)
- Controle de versão com Git e GitHub
- Qualidade de software (ISO 25010, métricas)

## Ferramentas

draw.io / Astah (UML), Git, GitHub, Jira / Trello (Scrum)
