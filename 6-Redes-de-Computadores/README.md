# 6. Redes de Computadores

Disciplina que aborda os fundamentos das redes de computadores, protocolos e comunicação de dados.

## Estrutura de Pastas

```
6-Redes-de-Computadores/
├── exercicios/    # Exercícios e anotações
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

- Modelo OSI (7 camadas) e modelo TCP/IP
- Endereçamento IP (IPv4, IPv6) e sub-redes (CIDR)
- Protocolos: TCP, UDP, HTTP/HTTPS, FTP, DNS, DHCP
- Equipamentos de rede: switch, roteador, firewall
- Redes sem fio (Wi-Fi 802.11)
- Segurança em redes: VPN, criptografia, firewall
- Simulação de redes com Cisco Packet Tracer

## Ferramentas

Wireshark, Cisco Packet Tracer, nmap
