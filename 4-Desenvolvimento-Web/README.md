# 4. Desenvolvimento Web

Disciplina que aborda as tecnologias e boas práticas para criação de aplicações web front-end e back-end.

## Estrutura de Pastas

```
4-Desenvolvimento-Web/
├── exercicios/    # Exercícios práticos de HTML, CSS e JS
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

### Front-end
- HTML5 semântico
- CSS3 (Flexbox, Grid, responsividade)
- JavaScript (ES6+): DOM, eventos, fetch/AJAX
- Frameworks: React / Vue (conforme ementa)

### Back-end
- Node.js / Express  ou  PHP / Laravel
- APIs REST
- Autenticação e sessões

## Ferramentas

VS Code, Live Server, Node.js, npm/yarn
