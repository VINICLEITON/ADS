Estrutura Básica de um Documento HTML

Este arquivo demonstra os elementos mínimos que toda página HTML deve ter. Abra o código-fonte (Ctrl+U ou F12) para ver os comentários explicativos.

Dica: o navegador IGNORA comentários HTML — eles existem apenas para documentar o código para desenvolvedores. Comentários são escritos assim: <!-- texto -->

Resumo da Estrutura

  1. <!DOCTYPE html> – declara HTML5 (deve ser a 1.ª linha)
  2. <html lang="pt-BR"> – raiz do documento, define o idioma
  3. <head> – metadados: charset, viewport, title, links CSS/JS
  4. <body> – conteúdo visível: textos, imagens, vídeos, formulários
