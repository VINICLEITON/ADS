3. Boas Práticas de Acessibilidade em Tabelas

  • Use <caption> para descrever o propósito da tabela.
  • Use <th scope="col"> nos cabeçalhos de coluna.
  • Use <th scope="row"> nos cabeçalhos de linha.
  • Use <thead>, <tbody> e <tfoot> para estruturar semanticamente.
  • Nunca use tabelas para layout de página — use CSS Flexbox/Grid.
