CSS – Box Model (Modelo de Caixa)

Todo elemento HTML é uma caixa com 4 camadas: content → padding → border → margin. Use o DevTools (F12 → aba "Computed") para visualizar o box model de qualquer elemento.

1. Content (Conteúdo)

200×60 px

2. Padding (Espaçamento Interno)

padding: 20px 40px

3. Border (Borda)

border: 3px solid + border-radius: 12px
4 lados com estilos diferentes

4. Margin (Espaçamento Externo)

margin: 20px auto

O elemento está centralizado com margin: 0 auto.

5. box-sizing: content-box vs border-box

content-box
border-box

6. Overflow

Texto muito longo que ultrapassa a altura definida de 60px e será ocultado.
Texto muito longo que ultrapassa a altura definida de 80px. Neste caso aparece barra de rolagem automática.
