Sobre o Projeto

Este projeto reúne exercícios e exemplos de ensino das disciplinas do curso de Análise e Desenvolvimento de Sistemas (ADS).


Os conteúdos cobrem os fundamentos de HTML, CSS e JavaScript, com código comentado e interativo para facilitar o aprendizado.


Cada tópico é um arquivo HTML autocontido, com explicações detalhadas, exemplos de código e demonstrações práticas.
