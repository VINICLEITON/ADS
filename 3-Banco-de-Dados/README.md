# 3. Banco de Dados

Disciplina que aborda modelagem, projeto e manipulação de bancos de dados relacionais e não-relacionais.

## Estrutura de Pastas

```
3-Banco-de-Dados/
├── exercicios/    # Exercícios práticos de SQL e modelagem
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

- Modelo Entidade-Relacionamento (MER / DER)
- Modelagem Relacional e Normalização (1FN, 2FN, 3FN)
- SQL: DDL (`CREATE`, `ALTER`, `DROP`)
- SQL: DML (`INSERT`, `UPDATE`, `DELETE`)
- SQL: DQL (`SELECT`, `JOIN`, `GROUP BY`, `HAVING`)
- Transações, Índices e Views
- Introdução a bancos NoSQL (MongoDB, Redis)

## Ferramentas

MySQL / PostgreSQL / SQLite / MongoDB
