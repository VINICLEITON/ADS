# 8. Matemática Discreta

Disciplina que aborda os fundamentos matemáticos essenciais para a Computação.

## Estrutura de Pastas

```
8-Matematica-Discreta/
├── exercicios/    # Exercícios resolvidos
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

- Lógica proposicional e predicados
- Conjuntos, relações e funções
- Indução matemática
- Combinatória (permutação, combinação, arranjo)
- Teoria dos grafos (caminhos, árvores, coloração)
- Álgebra Booleana e circuitos lógicos
- Aritmética modular e criptografia básica
- Recorrência e análise de complexidade

## Ferramentas

Python (sympy, numpy), LaTeX para notação matemática
