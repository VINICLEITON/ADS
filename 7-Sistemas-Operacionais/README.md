# 7. Sistemas Operacionais

Disciplina que aborda os conceitos fundamentais dos sistemas operacionais, seu funcionamento e administração.

## Estrutura de Pastas

```
7-Sistemas-Operacionais/
├── exercicios/    # Exercícios e scripts
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

- Conceitos de SO: núcleo (kernel), processos, threads
- Gerência de processos e escalonamento de CPU
- Gerência de memória: paginação, segmentação, memória virtual
- Sistemas de arquivos: FAT32, NTFS, ext4
- Gerência de dispositivos e drivers
- Segurança e controle de acesso
- Linha de comando Linux (bash) e Windows (PowerShell/CMD)
- Virtualização e containers (Docker)

## Ferramentas

Linux (Ubuntu/Debian), VirtualBox / VMware, bash, Docker
