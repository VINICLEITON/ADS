# 1. Algoritmos e Programação

Disciplina focada em lógica de programação, algoritmos e introdução à programação estruturada e orientada a objetos.

## Estrutura de Pastas

```
1-Algoritmos-e-Programacao/
├── exercicios/    # Exercícios práticos resolvidos
└── projetos/      # Projetos e trabalhos da disciplina
```

## Tópicos Abordados

- Variáveis, tipos de dados e operadores
- Estruturas de controle (`if/else`, `switch`)
- Laços de repetição (`for`, `while`, `do-while`)
- Funções e procedimentos
- Arrays e matrizes
- Introdução à Orientação a Objetos (OOP)

## Linguagens

Python / Java / C (conforme orientação do professor)
